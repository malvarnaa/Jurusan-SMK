Rekayasa Perangkat Lunak (RPL)

Deskripsi Jurusan

Jurusan Rekayasa Perangkat Lunak (RPL) mempelajari tentang pengembangan, pemeliharaan, dan manajemen perangkat lunak. Siswa akan dilatih menjadi programmer yang handal dan mampu membuat aplikasi yang berkualitas.

Program Belajar

  • Pemrograman Dasar
  • Pemrograman Web
  • Database
  • Pengembangan Aplikasi Mobile

Prospek Kerja

Lulusan jurusan RPL dapat bekerja sebagai programmer, web developer, mobile developer, atau software engineer di berbagai perusahaan teknologi dan startup.
